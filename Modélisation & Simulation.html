📐 CH1 — Modélisation des Systèmes & Processus Stochastiques

DÉFINITIONS FONDAMENTALES

Modèle & Système

Modélisation : Création d'une représentation simplifiée d'un système réel pour l'étudier, prédire ou simuler son comportement sans avoir à expérimenter sur le système réel.
Système : Ensemble d'entités interconnectées organisées pour accomplir un objectif. Ex: banque avec clients et caissiers.
État : Collection de variables nécessaires pour décrire le système à un instant t (variables d'état). Ex: nombre de clients dans une file.
Modèle descriptif : Décrit structure/comportement sans prédiction (flowchart). Analytique : équations mathématiques pour prédire (M/M/1).
DéterministeStochastiqueDiscret/Continu

PROCESSUS STOCHASTIQUE

Définition & Classification

Processus stochastique : {X(t), t ∈ T} — famille de variables aléatoires définies sur le même espace de probabilité, indexées par t (temps).
Classification selon (T, E) : T discret, E discret → CMTD T continu, E discret → CMTC T discret, E continu → VA discret-temps T continu, E continu → Diffusion (Brownien)
✅ Propriété de Markov : P(Xₙ=j | Xₙ₋₁, …, X₀) = P(Xₙ=j | Xₙ₋₁). Le futur ne dépend QUE du présent.

DISTRIBUTIONS CLÉS

Lois Utilisées en ModSim

Exponentielle Exp(λ) : fX(x)=λe^(-λx). E[X]=1/λ, Var=1/λ². SEULE loi continue sans mémoire. Modélise durées de séjour en CMTC.
Géométrique modifiée (p) : P[X=k]=(1-p)pᵏ, k≥0. E[X]=p/(1-p). Sans mémoire discrète.
Poisson(λt) : P[Z(t)=i]=e^(-λt)(λt)ⁱ/i!. Modélise le nombre d'arrivées dans [0,t]. Lié à l'exponentielle : si inter-arrivées~Exp(λ) → processus de Poisson.
E[X] = Σ x·P[X=x] (discret) Var(X) = E[X²] − (E[X])² C²ₓ = Var(X)/(E[X])² [coeff. variation]

PROPRIÉTÉ SANS MÉMOIRE

Preuve Exponentielle

Énoncé : P[X > s+t | X > s] = P[X > t]. Le temps déjà écoulé n'influence pas le temps restant.

PREUVE

P[X > s+t | X > s] = P[X > s+t ∩ X > s] / P[X > s]
= P[X > s+t] / P[X > s]
= e^(-λ(s+t)) / e^(-λs)
= e^(-λt) = P[X > t]  ✓

→ Seule loi exponentielle vérifie cela en continu
→ Seule loi géométrique en discret

TRANSFORMÉE DE LAPLACE

Outil Mathématique

Définition : LX(s) = E[e^(-sX)] = ∫₀⁺∞ e^(-sx) fX(x)dx. Transforme équations différentielles en équations algébriques.
LX(0) = 1 0 ≤ LX(s) ≤ 1 pour s ≥ 0 Indépendance: L_{X₁+X₂}(s) = LX₁(s)·LX₂(s) Moments: E[Xⁿ] = (-1)ⁿ L^(n)_X(0)
Ex: Exp(λ) → LX(s) = λ/(λ+s)
🔷 CH2 — Chaînes de Markov à Temps Discret (CMTD)

DÉFINITION CMTD

Structure & Matrice P

CMTD homogène : pᵢⱼ = P(Xₙ=j | Xₙ₋₁=i) ne dépend pas de n. Les probabilités de transition sont constantes dans le temps.
Matrice stochastique P = [pᵢⱼ] : • pᵢⱼ ≥ 0 pour tout (i,j) • Σⱼ pᵢⱼ = 1 pour tout i (lignes = 1) Analyse transitoire : π⁽ⁿ⁾ = π⁽⁰⁾ · Pⁿ Chapman-Kolmogorov : p⁽ⁿ⁾ᵢⱼ = Σₖ p⁽ᵐ⁾ᵢₖ · p⁽ⁿ⁻ᵐ⁾ₖⱼ
Accessibilité : i→j si ∃m≥1 tel que p⁽ᵐ⁾ᵢⱼ>0. Communication : i↔j si i→j ET j→i.

CLASSIFICATION DES ÉTATS

Transitoire vs Récurrent

fᵢᵢ = Σₙ₌₁^∞ f⁽ⁿ⁾ᵢᵢ = probabilité de retour en i Mᵢ = Σₙ₌₁^∞ n·f⁽ⁿ⁾ᵢᵢ = temps moyen de retour
TypeConditionInterprétation
Transitoirefᵢᵢ < 1Peut ne jamais revenir
Récurrentfᵢᵢ = 1Retour garanti (prob. 1)
Rée. non nulfᵢᵢ=1, Mᵢ<∞Retour fini
Rée. nulfᵢᵢ=1, Mᵢ=∞Retour infini (CMTD infinie)
✅ CMTD finie irréductible → tous les états sont récurrents non nuls.
fᵢⱼ = pᵢⱼ + Σₖ≠ⱼ pᵢₖ·fₖⱼ [prob. atteindre j] Mᵢⱼ = 1 + Σₖ≠ⱼ pᵢₖ·Mₖⱼ [temps moyen i→j]

PÉRIODE & ERGODICITÉ

Convergence CMTD

Période : D(i) = PGCD{k≥1 : p⁽ᵏ⁾ᵢᵢ > 0}. Si D(i)=1 → apériodique. Si D(i)>1 → périodique. Si pᵢᵢ>0 → D(i)=1 automatiquement.
CMTD irréductible : tous états communiquent. Une seule classe d'équivalence. Tous les états ont la même période.
État absorbant : pₖₖ=1. Classe fermée réduite à un état. Ex: ruine dans le problème du joueur.
✅ Ergodicité CMTD : finie + irréductible + apériodique → lim π⁽ⁿ⁾ = π existe, indépendant de π⁽⁰⁾.
Régime stationnaire : { π · P = π { Σⱼ πⱼ = 1 Propriété : πⱼ = 1/Mⱼ

NAISSANCE-MORT CMTD

Processus Infini

Transitions : p₀₁=1, pᵢ,ᵢ₊₁=p, pᵢ,ᵢ₋₁=q (∀i≥1). Condition d'ergodicité : p < q.

RÉSOLUTION

Équation de balance : πᵢ·q = πᵢ₋₁·p
→ πᵢ = (p/q)ⁱ · π₀

Normalisation (si p/q < 1) :
Σᵢ₌₀^∞ πᵢ = π₀ · Σᵢ(p/q)ⁱ = π₀/(1-p/q) = 1
→ π₀ = 1 - p/q

Distribution finale :
πᵢ = (1 - p/q) · (p/q)ⁱ  [Géométrique]
🟣 CH3 — Chaînes de Markov à Temps Continu (CMTC)

DÉFINITION CMTC

Matrice Q & Taux

CMTC : Processus de Markov à temps continu. Le temps de séjour dans chaque état i suit une loi Exp(µᵢ) sans mémoire. Les transitions se font par des taux µᵢⱼ.
Générateur infinitésimal Q = [qᵢⱼ] : qᵢⱼ = µᵢⱼ si i ≠ j qᵢᵢ = -Σⱼ≠ᵢ µᵢⱼ si i = j (diagonal négatif) RÈGLE FONDAMENTALE : Σⱼ qᵢⱼ = 0 pour tout i → Somme de chaque ligne de Q = 0
✅ CMTD incluse : pᵢⱼ = µᵢⱼ/µᵢ (normaliser les taux)

ÉQUATIONS D'ÉTAT

Chapman-Kolmogorov CMTC

Interprétation flux : La variation de probabilité d'être en i = flux entrant depuis tous les autres états − flux sortant vers tous les autres états.
dπᵢ(t)/dt = Σⱼ≠ᵢ µⱼᵢ·πⱼ(t) − πᵢ(t)·Σⱼ≠ᵢ µᵢⱼ ← flux entrant → ← flux sortant → Forme matricielle : dπ(t)/dt = π(t) · Q Solution : π(t) = π(0)·e^(Qt)
CMTC ergodique : CMTC finie + irréductible → toujours ergodique. Pas de problème de périodicité (temps exponentiel = apériodique par nature).
Régime stationnaire : { π · Q = 0 (différent de CMTD : πP=π) { Σᵢ πᵢ = 1

EXEMPLE CMTC

Système 3 États

CONSTRUCTION DE Q

États: 0, 1, 2. Taux: µ₀₁=2, µ₁₀=1, µ₁₂=3, µ₂₁=4

Q = [[-2,   2,   0 ],
     [ 1,  -4,   3 ],
     [ 0,   4,  -4 ]]

Vérification : chaque ligne somme à 0 ✓

Système πQ=0, Σπᵢ=1 :
-2π₀ + π₁ = 0           → π₁ = 2π₀
2π₀ - 4π₁ + 4π₂ = 0    → π₂ = (4π₁ - 2π₀)/4 = 3π₀/2
π₀ + π₁ + π₂ = 1
→ π₀(1 + 2 + 3/2) = 1  → π₀ = 2/9
→ π₁ = 4/9, π₂ = 3/9

🟠 CH4 — Files d'Attente & Processus de Naissance-Mort

PROCESSUS DE POISSON

Modèle d'Arrivées

Processus de Poisson de taux λ : événements aléatoires, indépendants, dans le temps. Si inter-arrivées ~ Exp(λ) et indépendantes → processus de Poisson.
P[Z(t)=i] = e^(-λt)·(λt)ⁱ/i! [loi Poisson] P[Tₙ > t] = e^(-λt) [inter-arrivée Exp] P[1 arrivée dans Δt] ≈ λ·Δt P[2+ arrivées dans Δt] ≈ 0 [infinitésimal]
✅ Superposition de Poisson : si X~Pois(λ₁) et Y~Pois(λ₂) indépendants → X+Y ~ Pois(λ₁+λ₂)

NOTATION DE KENDALL

A/B/s/K/N/D

A=loi arrivées (M=Markovien/Exp, D=Déterministe, G=Général). B=loi service. s=nb serveurs. K=capacité (∞/défaut). N=pop. source (∞/défaut). D=discipline (FIFO/défaut).

NAISSANCE-MORT CMTC

Modèle Général

Naissance-Mort : CMTC avec transitions uniquement vers i-1 et i+1. λᵢ = taux naissance (arrivée). µᵢ = taux mort (service). Modèle général des files M/M/*.
Récurrence stationnaire : µ₁π₁ = λ₀π₀ → π₁ = (λ₀/µ₁)π₀ πᵢ = π₀ · Πₖ₌₀ⁱ⁻¹ (λₖ/µₖ₊₁) π₀ = 1 / Σᵢ₌₀ᴺ Πₖ₌₀ⁱ⁻¹(λₖ/µₖ₊₁)
⚠️ Cas homogène λᵢ=λ, µᵢ=µ : ρ=λ/µ. πᵢ=ρⁱ·π₀. Stable seulement si ρ<1.

FILE M/M/1

Formules Complètes

ρ = λ/µ (doit être < 1) πᵢ = (1-ρ)·ρⁱ [géométrique] N = ρ/(1-ρ) = λ/(µ-λ) Nq = ρ²/(1-ρ) = λ²/[µ(µ-λ)] W = 1/(µ-λ) [temps dans système] Wq = ρ/(µ-λ) = λ/[µ(µ-λ)] Loi de Little : N = λ·W, Nq = λ·Wq
✅ Loi de Little : universelle, valide pour tout système stable, quelle que soit la distribution.

FILE M/M/1/K

Capacité Limitée

Clients arrivant quand le système est plein (état K) sont perdus. Toujours stable même si ρ≥1.
λᵢ = λ si i < K, λᵢ = 0 si i = K πᵢ = ρⁱ·π₀ pour i=0,…,K π₀ = (1-ρ)/(1-ρᴷ⁺¹) si ρ ≠ 1 π₀ = 1/(K+1) si ρ = 1 Prob. blocage = πK Débit effectif : λ_eff = λ·(1-πK) N = Σᵢ₌₀ᴷ i·πᵢ (calculer explicitement)
⚠️ Pour M/M/1/K avec ρ=1 : distribution uniforme, πᵢ=1/(K+1) pour tout i.

🔷 CMTD — Chaîne de Markov à Temps Discret

DÉFINITION & MATRICE

Structure CMTD

pᵢⱼ ≥ 0 et Σⱼ pᵢⱼ = 1 (∀i) π⁽ⁿ⁾ = π⁽⁰⁾ · Pⁿ p⁽ⁿ⁾ᵢⱼ = Σₖ p⁽ᵐ⁾ᵢₖ · p⁽ⁿ⁻ᵐ⁾ₖⱼ [Chapman-Kolmogorov]

CLASSIFICATION ÉTATS

Transitoire / Récurrent

fᵢᵢ < 1 → Transitoire fᵢᵢ = 1 → Récurrent fᵢᵢ = 1, Mᵢ < ∞ → Récurrent non nul fᵢᵢ = 1, Mᵢ = ∞ → Récurrent nul (CMTD ∞) fᵢⱼ = pᵢⱼ + Σₖ≠ⱼ pᵢₖ·fₖⱼ Mᵢⱼ = 1 + Σₖ≠ⱼ pᵢₖ·Mₖⱼ

PÉRIODE & ERGODICITÉ

Convergence

D(i) = PGCD{k≥1 : p⁽ᵏ⁾ᵢᵢ > 0} D(i) = 1 → apériodique pᵢᵢ > 0 → D(i) = 1 automatiquement CMTD ergodique ⟺ irréductible + apériodique

RÉGIME STATIONNAIRE

Distribution π

{ π · P = π { Σⱼ πⱼ = 1 πⱼ = 1/Mⱼ (inverse temps moyen retour) π = limite unique si CMTD ergodique

NAISSANCE-MORT CMTD

Infinie (p, q)

pᵢ,ᵢ₊₁ = p, pᵢ,ᵢ₋₁ = q, p+q=1 Ergodicité : p < q πᵢ = (1 - p/q) · (p/q)ⁱ [Distribution géométrique de raison p/q]
🟣 CMTC — Chaîne de Markov à Temps Continu

GÉNÉRATEUR Q

Matrice Infinitésimale

qᵢⱼ = µᵢⱼ si i ≠ j qᵢᵢ = -Σⱼ≠ᵢ µᵢⱼ si i = j RÈGLE : Σⱼ qᵢⱼ = 0 pour tout i (somme de chaque ligne = 0)

CMTD INCLUSE

Lien CMTD/CMTC

pᵢⱼ = µᵢⱼ / µᵢ où µᵢ = Σₖ≠ᵢ µᵢₖ (normaliser les taux de sortie) CMTC irréductible ⟺ CMTD incluse irréductible

ÉQUATIONS D'ÉTAT

Chapman-Kolmogorov

dπᵢ(t)/dt = Σⱼ≠ᵢ µⱼᵢπⱼ - πᵢΣⱼ≠ᵢ µᵢⱼ dπ(t)/dt = π(t) · Q

RÉGIME STATIONNAIRE

Distribution π

{ π · Q = 0 ← (≠ CMTD : πP=π) { Σᵢ πᵢ = 1 CMTC finie + irréductible → toujours ergodique (pas de problème de périodicité)
🟠 Files d'Attente

NAISSANCE-MORT GÉNÉRAL

Formule Récurrente

πᵢ = π₀ · Πₖ₌₀ⁱ⁻¹ (λₖ/µₖ₊₁) π₀ = 1 / Σᵢ₌₀ᴺ Πₖ₌₀ⁱ⁻¹(λₖ/µₖ₊₁) Homogène (λᵢ=λ, µᵢ=µ) : πᵢ = ρⁱ · π₀ avec ρ = λ/µ

FILE M/M/1

Capacité infinie

ρ = λ/µ < 1 (stabilité obligatoire) πᵢ = (1-ρ)·ρⁱ N = ρ/(1-ρ) Nq = ρ²/(1-ρ) W = 1/(µ-λ) Wq = ρ/(µ-λ) = λ/[µ(µ-λ)]

FILE M/M/1/K

Capacité K

π₀ = (1-ρ)/(1-ρᴷ⁺¹) si ρ ≠ 1 π₀ = 1/(K+1) si ρ = 1 πᵢ = ρⁱ · π₀ Prob. blocage = πK λ_eff = λ·(1-πK)

LOI DE LITTLE

Universelle

N = λ · W [système complet] Nq = λ · Wq [file d'attente seule] W = Wq + 1/µ [temps attente + service] Valide : tout système stable, toute distribution A/B quelconques.

PROCESSUS DE POISSON

Propriétés

P[Z(t)=k] = e^(-λt)·(λt)ᵏ/k! E[Z(t)] = λt, Var[Z(t)] = λt Inter-arrivées ~ Exp(λ), E=1/λ P[arrivée dans Δt] ≈ λΔt

DISTRIBUTIONS CLÉS

Rappels

Exp(λ) : E=1/λ, Var=1/λ² Géom(p): E=p/(1-p), Var=p/(1-p)² Pois(λ): E=λ, Var=λ C²=Var/E² [coeff variation]
